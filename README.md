# Crud Penggajian Karyawan Menggunakan Livewire

Aplikasi Sederhana ini untuk membuat stack backend tanpa endpoint menggunakan livewire



## Fitur
* SPA Dengan Livewire
* Auth Jetstream
* Livewire
* Laravel 8
* Unit Test
* Validasi Real-time
* Menggunakan Eloquent

## Tata Cara Penginstalan
* Clone Repo ini
* Composer Install
* NPM Install
* Setup .env databaseny
* php artisan serve
